About the Role

We are building a Hospital Super App that unifies teleconsultation, appointments, lab integrations, EHR connectivity, home care services, and corporate wellness into a single seamless mobile experience.
We are looking for a highly capable Full Stack Developer with strong expertise in React Native (mobile) and FastAPI (backend) who can contribute across the stack and work in a fast-paced, AI-accelerated development environment.


Key Responsibilities

Mobile App Development (Primary)

Design and develop scalable mobile applications using React Native for Android and iOS
Build reusable, maintainable, and high-performance UI components
Optimize app performance, responsiveness, and user experience
Handle offline-first features, push notifications, and real-time updates

Backend Development (Secondary but Important)

Develop REST APIs using FastAPI (Python)
Design scalable backend architecture hosted on AWS EC2
Integrate MongoDB for efficient data modeling and storage
Ensure API security, authentication (JWT/OAuth), and performance

System Integration

Integrate with external systems like:
Electronic Health Records (EHR)
Lab systems (LIS)
Payment gateways
Teleconsultation/video APIs
Work on healthcare workflows including appointments, patient records, and home care services

AI-Assisted Development

Use tools like Cursor / Claude Code / AI coding assistants to:
Accelerate development
Improve code quality
Automate repetitive tasks
Continuously improve development efficiency using AI tools

Collaboration & Ownership

Work closely with product managers, designers, and backend teams
Take ownership of modules from design to deployment
Participate in code reviews and maintain high coding standards
