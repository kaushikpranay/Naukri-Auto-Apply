Position Overview:
We are seeking a driven and technically capable Agentic AI Developer to join our growing AI Engineering team. In this mid-level role, you will contribute to the design and development of multi-agent systems, integrate agentic tooling into production pipelines, and help build intelligent automation solutions that power our next-generation cybersecurity and compliance platforms.

Key Responsibilities:
  • Develop and maintain agentic pipelines using LangChain and LangGraph, including tool-use chains, ReAct loops, and stateful multi-step reasoning workflows.
  • Build and extend Agent Harness components — defining tool schemas, agent personas, execution environments, and guardrail configurations.
  • Implement and maintain MCP (Model Context Protocol) server and client integrations for structured tool invocation across agent pipelines.
  • Develop Agent-to-Agent (A2A) communication flows — including task delegation, shared context management, and inter-agent messaging protocols.
  • Assist in fine-tuning language models using domain-specific datasets; support training pipeline setup and evaluation benchmarking
  • Integrate Claude SDK and other LLM provider APIs into agent systems with proper prompt engineering and response handling.
  • Build user-facing interfaces using React and HTMX to expose agentic capabilities with live streaming outputs and task dashboards.
  • Participate in architecture reviews, testing, and documentation of agent interaction maps and system behaviours.
  • To assist company from time to time

Required Skills and Qualifications:
  • 2–4 years in software or AI/ML engineering, with at least 1 year on agentic or LLM-powered systems.
  • Hands-on experience building agent pipelines with LangChain, LangGraph, or equivalent.
  • Working knowledge of model fine-tuning using Hugging Face, OpenAI, Anthropic, or similar.
  • Familiarity with vector databases (Qdrant, Pinecone, Chroma) for agent memory and retrieval.
  • Proficiency in Python; basic TypeScript knowledge is a plus.
