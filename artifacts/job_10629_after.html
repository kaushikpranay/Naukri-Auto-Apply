Job Description

Chistats is a data science company helping clients leverage data to accelerate and augment business value. With a team passionate about solving complex problems, we are offering internship opportunities across software engineering, including data science and data engineering. As an intern at Chistats, you will work closely with our data science, engineering, and analytics teams to support end-to-end project workflows. 

Selected intern's day-to-day responsibilites include: 

  • Assisting in data preparation, cleaning, validation, and transformation for various analytical and engineering tasks.
  • Contributing to machine learning workflows, including model building, evaluation, and exploratory data analysis.
  • Supporting development and maintenance of data pipelines, API integrations, and database operations.
  • Helping create dashboards, reports, MIS sheets, and analytical summaries for business insights.
  • Participating in backend and frontend development tasks, working with REST APIs, UI enhancements, and web applications.
  • Assisting in building and operating CI/CD pipelines, cloud environments, and containerized deployments.
  • Troubleshooting issues, maintaining documentation, collaborating with cross-functional teams, and contributing to prototype or feature development.

Other requirements : 

  • Strong analytical thinking, problem-solving, and attention to detail.
  • Basic working knowledge of Python, SQL, or JavaScript.
  •  Understanding of data handling: cleaning, preparation, validation, and visualization.
  •  Good communication skills with ability to document work clearly.
  • Willingness to learn new technologies and adapt to fast-paced projects.
  • Exposure to Python libraries like NumPy, Pandas, Scikit-learn (good to have).
  •  Familiarity with TensorFlow, PyTorch, or similar ML frameworks (good to have).
  • Knowledge of ETL/ELT, API integrations, or relational databases (PostgreSQL/MySQL/Snowflake) is an additional plus.
  • Good understanding of Excel (charts, pivots, formulas).
  •  Familiarity with Power BI, Tableau, or other visualization tools (nice to have).
  • Basic exposure to statistics, reporting, or performance metrics is an additional advantage.
  • Exposure to frontend technologies like React or Angular (good to have).
  • Understanding of backend frameworks like FastAPI, Flask, Django is an additional plus.
  •  Familiarity with HTML, CSS, Git, and responsive UI concepts.
  • Familiarity with Linux, shell scripting, or cloud platforms (AWS/Azure/GCP) is good to have.
  •  Exposure to Docker, CI/CD pipelines (GitHub Actions/Jenkins), or container orchestration is an added advantage.
  • Understanding of monitoring, logging, or environment setup is an extra plus.
